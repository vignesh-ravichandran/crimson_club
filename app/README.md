# App (Next.js App Router)

- **`(auth)/`** — Sign-in, sign-up; no bottom nav.
- **`(app)/`** — Main app: Home, Journeys, Review, Profile; layout with bottom nav.
- **`api/`** — API route handlers (auth, users, journeys, daily, goals, weekly reviews, lessons, leaderboard, home). See [docs/lld/apis-and-modules.md](../docs/lld/apis-and-modules.md) and [docs/lld/api-contracts.md](../docs/lld/api-contracts.md).
- **`layout.tsx`** — Root layout (at repo root under `app/`).

Route map: [docs/lld/frontend-design.md](../docs/lld/frontend-design.md) §8.
