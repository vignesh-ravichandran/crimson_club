# Components

- **`ui/`** — Primitives (Button, Card, Chip, Input, Label, Tabs, Divider). Token-driven; see [docs/design/colour-palette.md](../docs/design/colour-palette.md).
- **`layout/`** — AppShell, BottomNav, PageHeader, Section.
- **`domain/`** — Feature components (JourneyCard, DimensionRow, WeeklyReviewCard, GoalCard, LessonItem).
- **`forms/`** — Shared form patterns if needed.

Conventions: [docs/lld/frontend-design.md](../docs/lld/frontend-design.md) §4, §7.
